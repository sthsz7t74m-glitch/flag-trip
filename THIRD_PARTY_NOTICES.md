# Third-party data notices

## Country data

`src/main/resources/static/data/countries.json` contains selected and transformed fields from
[world-countries](https://www.npmjs.com/package/world-countries) 5.1.0.
The source database is distributed under the
[Open Database License (ODbL) 1.0](https://opendatacommons.org/licenses/odbl/1-0/).

## Japanese capital labels

Japanese capital labels were obtained from [Wikidata](https://www.wikidata.org/) using its SPARQL endpoint.
Wikidata structured data is available under [CC0 1.0](https://creativecommons.org/publicdomain/zero/1.0/).

## Flag images

The application displays flag images from [FlagCDN](https://flagcdn.com/) and falls back to Unicode flag emoji when an image cannot be loaded.
FlagCDN is based on the [flag-icons](https://github.com/lipis/flag-icons) project.
